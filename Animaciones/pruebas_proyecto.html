Animación CSS: Lobo Persiguiendo a la Niña

Para usar tus sprites reales:

Reemplaza los estilos .girl y .wolf con:

.girl {
  position: absolute;
  width: [ancho-del-sprite]px;
  height: [alto-del-sprite]px;
  background: url('lobo-persiguiendo-niña.png') no-repeat;
  animation: runGirl 3s linear infinite;
}

.wolf {
  position: absolute;
  width: [ancho-del-sprite]px;
  height: [alto-del-sprite]px;
  background: url('lobo-corriendo.png') no-repeat;
  animation: runWolf 3s linear infinite;
}
